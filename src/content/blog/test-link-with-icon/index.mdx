  // 代码块也能正常工作

</Folding>

<Folding title="默认展开的折叠面板" open>
  这是一个默认展开的折叠面板，通过设置 `open` 属性为 `true` 实现。
  
  - 这个面板在页面加载时就会自动展开
  - 用户可以点击标题将其收起
  - 可以用于展示重要信息或经常访问的内容
  
  ```javascript
  // 使用示例

    内容
  </Folding>
  ```
</Folding>
